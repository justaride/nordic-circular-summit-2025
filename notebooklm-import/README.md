## Adding New Content

When adding new NotebookLM outputs:

1. Place files in appropriate subfolder (audio/, video/, documents/, visuals/)
2. Update manifest.json with metadata
3. Add entry to The Notebook page (`/app/notebook/page.tsx`)

## Manifest Schema

```json
{
  "id": "unique-slug",
  "title": "Display Title",
  "description": "Brief description",
  "type": "audio|video|pdf|image",
  "filename": "Original_Filename.ext",
  "dateGenerated": "2025-12-04",
  "duration": "~10 min",
  "topics": ["Topic1", "Topic2"],
  "sourceDocuments": ["doc1.md", "doc2.md"],
  "batch": 1
}
```
